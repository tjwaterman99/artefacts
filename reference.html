classmethod load()¶

Load and deserialize the artifact.

The deserialized artifact is cached, so calling this function twice will not update the artifact if it has been rebuilt.

In general, loading the artifacts is not necessary, and most tasks can be accomplished by using the API module instead of working with the artifact classes directly.

classmethod name()¶

The name of the artifact in snake_case.

classmethod path()¶

The path to the artifact.

The path is determined by the Configuration settings. By default artefacts will look in the ./target directory of the current working directory.
